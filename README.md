# LibraryWebApplication
<ul>
<li>UI: Vaadin 8</li>
<li>Доступ к данным: JDBC</li>
<li>Сервер: MySQL 5.7</li>
</ul>  
*Настройка и запуск

Импортировать проект (как проект maven)

Ввести в терминале
```
mvn install
cd target
java -jar demo-0.0.1-SNAPSHOT.jar
```
В браузере открыть
```
http://localhost:8080
```
